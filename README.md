# invisible_cloak

## How to run the invisible cloak code in visual studio code

1. create virtual environment

```bash
$ conda create -n hp
$ conda activate hp
```

2. install the packages

```bash
$ conda install pip numpy
$ pip install opencv-python
```

3. run the program

```bash
$ python main.py
```
